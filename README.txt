Minimal Responsive Pricing Table plugin helps you to create clean and responsive pricing tables with multiple layouts

== Description ==
Minimal Responsive Pricing Table plugin helps you to create clean and responsive pricing tables with multiple layouts. Minimal Responsive Pricing Table have all those features which you need to build a super clean and responsive pricing table for your website.

== Live demo ==

[See Live Demo and documentation here](http://plugins.digitsol.co/free/minimal-responsive-pricing-table/) 


= Features =
* Each pricing table will have it's own shortcode
* Re-Order plans via drag & drop
* Heading color and background is customize-able
* Pricing section color and background is customize-able
* Button color and background is customize-able
* Easy to use
* Documented with images & tool-tip
* 4 different responsive layouts
* Responsive Tables
* Make any plan \'Recommended\'


== Installation ==
1- Upload minimal pricing table plugin to the /wp-content/plugins/ directory
2- Activate the plugin through the \'Plugins\' menu in WordPress
